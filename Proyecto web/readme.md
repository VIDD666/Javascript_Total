# Gestor de Proyectos - Gestiones Antonella

## Descripción
Este proyecto es un Gestor de Proyectos realizado para el curso de Desarrollo Web en el Colegio Técnico Profesional San Rafael de Alajuela, Costa Rica. Permite administrar proyectos con funcionalidad para crear, listar, actualizar y eliminar proyectos mediante una API REST.

## Tecnologías Utilizadas
- Node.js
- Express
- JavaScript (frontend y backend)
- HTML y CSS
- Postman o REST Client para pruebas de API

## Instalación y Uso
